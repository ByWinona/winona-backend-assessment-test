# Winona Backend Challenge

## Setup

This is a NestJS application. 

Please clone the repository and run 
1. `npm install` to install dependencies
2. `npm run start:dev` to start the development server

The server will start on `http://localhost:3000`

## Stack 
This is a NestJS application using TypeORM with a SQLite database.
It's publishing an OpenAPI Specification available at `http://localhost:3000/api` when the server is running.

## Tasks

Add API endpoints to manage patients.
A patient can have prescriptions of medications.

Document the API through an OpenAPI Specification.
